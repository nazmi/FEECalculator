API 14E Eq. 2.14
HOW VELOCITY IS CALCULATED FROM FLOWRATE
v = Q_flowing / A_pipe  |  Q_flowing = Q_std × (P_std/P) × (T/T_std) × Z  |  Each pipe size line has slope = (P_std/P × T/T_std × Z) / A
Lines are linear because at fixed P & T, velocity ∝ flowrate. Slope is flatter for larger pipes (larger A). Changing P shifts all slopes — lower P means gas expands more → higher velocity for same MMscfd → why choke outlet is the critical sizing point.
ASME B31.3 Eq. 2.19 — Required wall thickness by pipe size
t = Pi·Do / (2·(S·E + Pi·Y))
t_required = pressure design thickness (in) · t_total = t_required + corrosion allowance · Schedule shown = minimum acceptable schedule (Sch 40/80/160/XXH)
Y = 0.4 (ferrous, T < 900°F, t < D/6) · E = weld joint factor · S = allowable stress (psi) · API 14E §2.6: min 0.050 in corrosion/mechanical allowance for carbon steel
API RP 14E §2.7 — JOINT CONNECTIONS
Hydrocarbon piping ≥2" diameter → butt welded. Piping ≤1½" → socket welded for: hydrocarbon service above ANSI 600#, above 200°F, subject to vibration, or glycol service. Wellhead piping is typically ANSI 900# or 1500# class.
API RP 14E §2.8 — THERMAL EXPANSION
Topsides wellhead piping at high temperature — check expansion criterion: (D × ΔL) / U² ≤ 0.03 per ANSI B31.3. ΔT typically 100–200°F for wellhead piping. Short runs generally self-compensating; L-bends and Z-offsets preferred over expansion bellows per API 14E.
